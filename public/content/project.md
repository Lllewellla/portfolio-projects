## Метаданные проекта

- **Название:** Дизайн станка плазменной резки Гиперплазма: как задача «сделайте красиво» превратилась в структурированное решение
- **Клиент:** Гиперплазма
- **Продукт:** Станок плазменной резки металла
- **Картинка:** https://optim.tildacdn.com/stor6266-3430-4462-a138-653238346664/-/format/webp/1b026ba925beb5691f6eedfd3410c412.png.webp
- **Срок:** 63 дня
- **Объём:** промышленный дизайн станка плазменной резки под бренд Гиперплазмы: консоль, корпусирование, цветографика и сценарии работы

## Артефакты (ноды на орбите)

Каждый артефакт задаётся заголовком и текстом. В заголовке через точку с запятой укажи: **тип**, **стейкхолдер**, **вес (1–5)**.

Допустимые типы: `quote`, `insight`, `constraint`, `pivot`, `task`.  
Стейкхолдеры: `client`, `marketing`, `production`, `finance`, `legal`, `users`, `internal`.

### quote ; client ; 5

«Со станком всё хорошо технически, но он не выглядит дорого» — так сформулировали исходную боль по виду станка.

### quote ; client ; 4

«Синий — цвет китайского производителя» — текущая цветовая раскладка покупных узлов выглядела «чужой» для бренда.

### quote ; client ; 4

«Основные цвета нашей компании это RAL 7016 и RAL 3020…» — запрос сделать так, чтобы станок говорил на языке бренда Гиперплазмы.

### constraint ; production ; 5

Часть узлов — покупные элементы: их габариты и цвет менять нельзя, а перекраска рельс возможна, но не сразу.

### constraint ; production ; 4

Все кожухи сгибаются из листового металла с радиусами 1,6–8 мм — сложные скульптурные формы не проходят по технологии.

### constraint ; production ; 3

Брендинг допускается только в понятных зонах: передние и задние торцы ванны, электрические шкафы, согласованные панели.

### insight ; internal ; 4

Рабочий чат по проекту — это карта принятия решений: 63 дня, 357 сообщений, 7 участников и 10+ итераций.

### insight ; users ; 4

Консоль управления должна выдерживать привычное поведение оператора: облокачивание, работу в перчатках, сенсорный экран с дублированием мышью и клавиатурой.

### insight ; internal ; 5

Задача «сделать, чтобы выглядело дорого» разворачивается в конкретные рычаги: цветографика, корпусирование, консоль, ограждения, системы хранения и подсветка.

### pivot ; internal ; 4

Команда делит работу на два связных направления: язык фирменной цветографики и язык формы/корпусирования станка.

### task ; internal ; 3

Подготовить подробный протокол встречи: зафиксировать боль «не выглядит дорого» и реальные технологические ограничения.

### task ; internal ; 3

Провести быструю дизайн‑аналитику: разложить, как работает станок от загрузки листа до выгрузки деталей, и собрать референсы «дорогого вида».

### insight ; marketing ; 4

Фирменные цвета RAL 7016 и RAL 3020 лучше работают как акценты на ключевых узлах — ванна, портал, консоль — а не как сплошная заливка всего станка.

### pivot ; production ; 4

Не бороться с базовым цветом покупных элементов, а строить фирменный образ вокруг поверхностей, которыми владеет сам заказчик.

### insight ; client ; 5

Решения по форме и цветографике можно масштабировать на линейку станков Гиперплазмы — это база для серийного продукта, а не единичный шоу‑образец.
